HackVM

A Hack virtual machine on simulated Hack hardware in your browser

Press "Load" to select some .VM files to run. All Hack OS functions are already included, and non .vm files will be ignored, so you can compile a folder by selecting all the files in that folder. Any infinite loops in the .vm code will also hang your browser, so be sure to test your software before running!
